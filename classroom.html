Refreshing crucial tools for teachers

The story of GitHub Classroom

Role

Program Manager, Experience Designer

Timeline

December 2019 - March 2020

Deliverables

  • UX teardown on the original Classroom site
  • New copy
  • Implementation of new design

Overview

GitHub Classroom is an automated tool that helps teachers manage and distribute assignments in a scalable way. Classroom has been improving for years, but some features key to classroom management, like autograding and an effective way to give feedback, were missing.
As autograding and feedback were planned for release, the landing page needed to reflect these changes, as well as be improved to fit the look of the other GitHub Education products.

The Problem

Teachers new to GitHub Classroom don't know how the product will meet their classroom management needs after visiting the website.

The landing page lacks key information about the product and is difficult to read and navigate.

The Process

We needed to know more about what our users were saying, so this project began with extensive user research carried out by a teammate. While my coworker learned crucial information from our users, I performed a UX teardown on the existing landing page.

Creating the landing page of teacher's dreams

The visual redesign of the landing page was led by another teammate, whom I supported with feedback and the implementation of her design with HTML and CSS (Using GitHub's Primer). The new landing page encompassed the teacher-directed copy I wrote after the teardown, as well as some more modern design elements to make the page pop, overall creating a more cohesive theme for products in the GitHub Education universe.
